This is a simple AI Chatbot application where users can log in and chat with an intelligent chatbot.  
The chatbot uses **Grok AI** to understand user messages and generate smart responses.  
User login is handled securely using **Firebase Authentication**, and basic user data is stored in **MongoDB**.

Users can:
- Log in securely
- Send messages to the chatbot
- Receive AI-generated replies in real time

Technologies used:
- Grok AI (chatbot intelligence)
- Firebase Authentication (login)
- MongoDB (database)
- Node.js & Express.js (backend)
- HTML, CSS, JavaScript (frontend)

This application is designed to be simple, fast, and easy to extend for future features like chat history, voice chat, or multi-language support.
